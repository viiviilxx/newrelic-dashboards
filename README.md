# newrelic-dashboards

A collection of JSON files for New Relic custom dashboards.

## Dashboards

| Dashboard Name | Description | Link |
| --- | --- | --- |
| Cloudflared | A dashboard to visualize and monitor the traffic and operational status of cloudflared (Cloudflare Tunnel). | [cloudflared.json](./cloudflared.json) |
| Nature Remo | A dashboard to monitor Nature Remo sensor data (temperature, humidity, illumination, etc.) and device status obtained via API integration. | [nature-remo.json](./nature-remo.json) |
